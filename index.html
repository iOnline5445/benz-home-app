📚 วิธีตั้งค่า (ฟรีทั้งหมด)
Facebook Graph API
ไปที่ developers.facebook.com → สร้าง App → ไปที่ Graph API Explorer → สร้าง User Token ด้วย permission: public_content
Firebase (ฟรี Spark Plan)
console.firebase.google.com → New Project → Firestore Database → Start in test mode → เอา API Key จาก Project Settings
Google Drive API
console.cloud.google.com → Enable Drive API → OAuth 2.0 → สร้าง Folder และเอา ID จาก URL
Claude API Key
console.anthropic.com → API Keys → สร้าง key ใหม่ (มี Free tier $5)
🔄 Workflow Architecture
// ขั้นตอนอัตโนมัติ
1. รับ FB URL → Extract Post ID
2. FB Graph API → ดึงข้อมูลโพสต์
3. Claude AI → วิเคราะห์อสังหาฯ
4. Claude AI → สร้าง Social Post
5. Firestore → บันทึก Database
6. Google Drive → อัพโหลดไฟล์
// ข้อมูลที่สกัด
• ประเภท/ราคา/ขนาดพื้นที่
• ที่ตั้ง/จังหวัด/ทำเล
• รายละเอียดห้อง/สิ่งอำนวยฯ
• ข้อมูลติดต่อผู้โพสต์
• URL รูปภาพทั้งหมด
• timestamp / source URL
