Export ข้อมูลทั้งหมด (ทรัพย์สิน + Agent + ลูกค้า + Users) เป็นไฟล์ JSON เพื่อสำรองข้อมูล
🔄 Real-time Sync
ข้อมูลอัพเดทอัตโนมัติทันทีเมื่อมีการเปลี่ยนแปลง — ไม่ต้องกด Sync
📅 Auto Backup ตามเวลา
ระบบสำรองข้อมูลอัตโนมัติ — ส่งอีเมลหรือดาวน์โหลดไฟล์ตามกำหนดเวลา
🔗 ตั้งค่า EmailJS (ส่งอีเมลอัตโนมัติ)
สมัครฟรีที่ emailjs.com → สร้าง Service + Template → กรอก ID ด้านล่าง
💡 Template variables ที่ใช้ได้: {{to_email}}, {{subject}}, {{backup_data}}, {{asset_count}}, {{agent_count}}, {{customer_count}}, {{backup_time}}
💡 ทางเลือก: Google Drive Backup
วิธีง่ายที่สุดสำหรับ Google Drive คือ:
1. กด Backup ทั้งหมด (.json) → ได้ไฟล์ benzhome_backup_xxxx.json
2. อัพโหลดไฟล์ขึ้น Google Drive ด้วยมือ หรือ
3. ใช้ Google Drive Desktop App ที่ sync โฟลเดอร์ Downloads อัตโนมัติ
ยังไม่เคย Backup
