📅 Auto Backup ตามกำหนดเวลา

ส่งไฟล์สำรองข้อมูลไปยัง Google Drive ของคุณโดยอัตโนมัติตามช่วงเวลาที่กำหนด

💾 ตั้งค่าสำรองข้อมูลไป Google Drive
กรอกลิงก์ Google Apps Script Web App เพื่อส่งไฟล์ไปเก็บบน Google Drive ของคุณโดยอัตโนมัติ:
📝 วิธีรับลิงก์ Google Apps Script (คลิกเพื่อดูขั้นตอน)
  1. ไปที่ Google Apps Script และสร้างโครงการใหม่
  2. คัดลอกโค้ดนี้ไปวาง: 
    function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var folderId = "YOUR_FOLDER_ID_HERE"; // แก้ไขเป็น ID โฟลเดอร์ Google Drive ของคุณ
    var folder = DriveApp.getFolderById(folderId);
    var file = folder.createFile(data.filename, data.content, "application/json");
    return ContentService.createTextOutput(JSON.stringify({status: "success", fileId: file.getId()}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({status: "error", message: err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
  3. คลิก Deploy (การทำให้ใช้งานได้) > New deployment (การทำให้ใช้งานได้ใหม่)
  4. เลือกประเภทเป็น Web app (เว็บแอป)
  5. ตั้งค่า: Execute as = Me (ฉัน) และ Who has access = Anyone (ทุกคน)
  6. คลิก Deploy และคัดลอก Web app URL นำมาวางในช่องด้านบน
